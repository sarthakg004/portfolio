# Assets Directory

Place your images in the following structure:

```
assets/
├── profile.jpg          # Your profile photo
├── projects/            # Project thumbnails
│   ├── fraud-detection.jpg
│   ├── ocr-engine.jpg
│   └── machine-translation.jpg
└── blogs/               # Blog post thumbnails
    ├── transformers.jpg
    ├── ocr-systems.jpg
    └── ml-deployment.jpg
```

## Recommended Image Sizes

- **Profile Image**: 400x400px (square, will be cropped to circle)
- **Project Thumbnails**: 800x500px (16:10 aspect ratio)
- **Blog Thumbnails**: 800x450px (16:9 aspect ratio)

## Placeholder Images

If you don't have images yet, the components will display gradient placeholders automatically.
